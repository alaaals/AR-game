Brief overview of the design process:


To implement a stack-like feature, I ensured that once a block has been collected by the user, it gets added into the "stacked blocks" array, which is then attatched to the Hiro tracker. I gave each block a (randomly generated) color and speed. The game goes throigh multtiple states: a Begin state, which is this one, a Play state, and then either a Win state (if player wins) or a Lose state (if player loses). Enjoy!
